# 🚀 Prajwal M — Developer Portfolio

> A sleek, dark-themed personal portfolio website showcasing my projects, skills, and journey as a Full-Stack Developer and CS & Engineering student at NIE, Mysuru.

---

## ✨ Features

- 🎬 Animated loading screen with progress bar
- 🌑 Dark UI with purple/violet accent theme
- 🔮 3D animated glowing orb on hero section
- 💡 Scroll-triggered reveal animations
- 🛠 Skills section with animated progress bars
- 📁 Featured projects with tech stack tags
- 📬 Contact form + info cards
- 💬 AI chat widget (bottom-right)
- 📱 Fully responsive design

No build step needed — just open `index.html` in your browser.

```bash

cd portfolio
open index.html
```

Or deploy instantly on **GitHub Pages**, **Netlify**, or **Vercel**.
